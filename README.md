# Recipe Manager

Проект для работы с рецептами, ингредиентами и списком покупок.

## Описание

В проекте реализованы классы:

- `Ingredient` — ингредиент с названием, количеством и единицей измерения;
- `Recipe` — рецепт с набором ингредиентов;
- `ShoppingList` — список покупок, который объединяет ингредиенты из разных рецептов;
- `DietaryRecipe` — рецепт с указанием диетической категории.

## Установка

Склонируйте репозиторий:

```bash
git clone <ссылка-на-репозиторий>
cd recipe-manager
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Запуск тестов:

```bash
pytest
```
